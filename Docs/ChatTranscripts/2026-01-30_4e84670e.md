## Game Center & Leaderboards

**Good idea, but scope it carefully.** Game Center integration is straightforward on iOS and adds real retention value. However:

- **Country flags**: Game Center doesn't natively expose player country. You'd need players to self-select, which adds friction. I'd skip this initially â€” Game Center already shows regional leaderboards natively.
- **Per-level leaderboards**: This makes sense for campaign mode. Game Center supports multiple leaderboard IDs, so one per planet is feasible.
- **Recommendation**: Implement a single global leaderboard first (classic mode), then add per-planet boards with the campaign update.

## Platform Difficulty (A/B/C) with Multipliers

**This is one of the strongest ideas here.** It adds replayability with zero new content â€” just positioning and multipliers. Some critique:

- **Multipliers (1x, 2x, 5x)**: The jump from 2x to 5x is steep. Consider 1x / 2.5x / 5x, or let playtesting decide. The 5x on platform C will dominate leaderboards â€” that's fine if intentional, but it means anyone chasing high scores *must* land C.
- **Fixed positions with drop from upper-left**: Good. The asymmetry (dropping left, hardest platform right) creates a natural skill curve. Platform C forces a full horizontal traverse with fuel cost â€” that's elegant design.
- **Visual distinction**: Make sure platforms are visually distinct (size, color, markings) so the risk/reward is immediately readable.

## Campaign Mode + Lives System

**Mixed feelings here.** The star system tied to platforms (A=1 star, B=2, C=3) is clean and intuitive â€” very Angry Birds, players understand it instantly.

**The lives system concerns me:**

- Lives create **frustration loops** in a physics game where crashes feel unfair (one bad tilt and you're done). Angry Birds works with lives because each attempt is 5 seconds. Your game has longer runs with more investment per attempt.
- **3 lives with game over = restart from where?** If it's restart the whole campaign, that's punishing. If it's just retry the level, lives don't add much over a simple retry button.
- The "3 stars = extra life, max 5" mechanic adds complexity without proportional fun. Players grinding 3-star landings for an extra life will feel like a chore, not a reward.

**My recommendation**: Drop lives entirely. Let players retry freely. The star system and leaderboards provide enough motivation. If you want consequences for crashing, consider a **fuel penalty** on retry (start with 90% fuel) or a **score streak** that resets on crash. These punish failure without blocking progress.
